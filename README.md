# MyStudyApp

这是一个多功能学习助手应用程序，旨在帮助用户提高学习效率。应用包含多个实用功能模块，如备忘录、计算器、计时器、闹钟和网页浏览器等。

## 功能特点

- **备忘录 (Memo)**: 记录学习笔记和待办事项。
- **计算器 (Calculator)**: 提供基础的数学计算功能。
- **计时器 (Chronometer)**: 用于计时学习和休息时间。
- **闹钟 (Alarm Clock)**: 设置提醒以帮助管理学习时间。
- **网页浏览器 (Web Browser)**: 快速访问常用学习网站。

## 模块介绍

- **MainActivity**: 主界面，提供各个功能模块的入口。
- **MemoActivity**: 备忘录功能，允许用户输入和保存笔记。
- **CalculatorActivity**: 实现基本的计算器功能，支持加减乘除运算。
- **AlarmClockActivity**: 设置和取消闹钟提醒。
- **WebActivity**: 内置浏览器，支持访问预设和自定义网站。
- **TimeUpActivity**: 计时器结束时的提醒界面。

## 技术栈

- **Android SDK**: 使用 Java 编写，基于 Android 平台。
- **Android Jetpack 组件**: 如 `Chronometer` 和 `AlarmManager`。
- **UI 组件**: 使用 `TextView`, `Button`, `EditText` 等构建用户界面。

## 安装与运行

1. 确保你已经安装了 Android Studio 和 Android SDK。
2. 下载项目并导入到 Android Studio。
3. 连接 Android 设备或使用模拟器。
4. 点击 "Run" 按钮构建并运行应用。
